#  back_end

> A node.js + express project

## 1.功能

```
- 登录 / 注销

- 接口
  - 页面权限
  - 指令权限
  - 权限配置
  - 二步登录

- 全局功能
  - 多种动态换肤
  - 动态侧边栏（支持多级路由嵌套）
  - 动态面包屑
  - 自适应收缩侧边栏

- 表格
  - 学生管理列表
  - 作业管理列表

- 错误页面
  - 401

- 組件
  - 分页
  - 列表

- 错误日志
- 
- 
```



## 2.开发

``` bash
# 克隆项目
git clone https://github.com/myGre/back_end.git

# 进入项目
cd back_end
# 安装依赖
npm install --registry=https://registry.npm.taobao.org

# serve with hot reload at localhost:3000 启动服务器3000
npm start


```
